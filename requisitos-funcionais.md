**Controle e Movimentação**

— O sistema deve permitir que o jogador mova o personagem em todas as direções em um mundo aberto

— O sistema deve permitir ações de escalada, corrida, agachamento e natação

— O sistema deve permitir que o jogador monte e desmonte cavalos

**Combate**

— O sistema deve oferecer combate corpo a corpo com diferentes posturas de luta

— O sistema deve permitir ataques furtivos e assassinatos silenciosos

— O sistema deve disponibilizar armas secundárias como arco, kunais, bombas e gancho

— O sistema deve permitir duelos individuais com mecânica de confronto direto

**Progressão e RPG**

— O sistema deve permitir evolução de habilidades através de uma árvore de talentos

— O sistema deve oferecer equipamentos e armaduras customizáveis com atributos diferentes

— O sistema deve registrar o progresso do jogador e salvar automaticamente

**Exploração e Mundo Aberto**

— O sistema deve gerar eventos climáticos dinâmicos (chuva, vento, neblina)

— O sistema deve oferecer um sistema de navegação por vento no lugar de minimapa tradicional

— O sistema deve disponibilizar atividades secundárias como santuários, banhos quentes e bambus de escrita

— O sistema deve permitir interação com NPCs e aceitar missões secundárias (Contos)

**Narrativa**

— O sistema deve apresentar cutscenes e diálogos que avançam a história principal

— O sistema deve oferecer escolhas que influenciam relações entre personagens

**Configurações**

— O sistema deve permitir ativar o Modo Kurosawa (preto e branco)

— O sistema deve oferecer dublagem em japonês e inglês

— O sistema deve permitir ajuste de dificuldade
